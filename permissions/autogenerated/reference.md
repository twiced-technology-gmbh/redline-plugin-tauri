## Default Permission

Default permissions for the Redline annotation overlay plugin

#### This default permission set includes the following:

- `allow-capture-screenshot`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`redline:allow-capture-screenshot`

</td>
<td>

Enables the capture_screenshot command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`redline:deny-capture-screenshot`

</td>
<td>

Denies the capture_screenshot command without any pre-configured scope.

</td>
</tr>
</table>
